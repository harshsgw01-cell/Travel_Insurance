## About travel_insurance

travel_insurance is a web application framework with expressive, elegant syntax. We believe development must be an enjoyable and creative experience to be truly fulfilling. travel_insurance takes the pain out of development by easing common tasks used in many web projects, such as:

- [Simple, fast routing engine](https://travel_insurance.com/docs/routing).
- [Powerful dependency injection container](https://travel_insurance.com/docs/container).
- Multiple back-ends for [session](https://travel_insurance.com/docs/session) and [cache](https://travel_insurance.com/docs/cache) storage.
- Expressive, intuitive [database ORM](https://travel_insurance.com/docs/eloquent).
- Database agnostic [schema migrations](https://travel_insurance.com/docs/migrations).
- [Robust background job processing](https://travel_insurance.com/docs/queues).
- [Real-time event broadcasting](https://travel_insurance.com/docs/broadcasting).

travel_insurance is accessible, powerful, and provides tools required for large, robust applications.
